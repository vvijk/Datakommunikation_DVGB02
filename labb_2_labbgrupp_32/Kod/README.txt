1. Use makefile with the command "make" to compile the code
2. Type "./sim" followed by 5 arguments in the terminal. For example "./sim 10 0.2 0.2 10 2"
3. The arguments in order represent: 1. Number of packets 2. Package Loss 3. Package corruption 4. Interval 5. Trace
4. Run the given program so see the packets travel from host A to host B, and depending on the chosen trace, different information will appear between each package being sent.
